## What we are building
We are building a drinking guide chat window for a pub, that is supposed to be a single page application with a user input at the bottom and the ai answer displayed in chat.
The colors should be reminiscent of an irish pub. the chat window has to be a wooden outside color but with a background of a color that is a light yellowish golden paper.

Stack to use:

- Next JS
- Tailwind css v4
- lucide react for icons

Name: "The Londoner" Drink Guide
